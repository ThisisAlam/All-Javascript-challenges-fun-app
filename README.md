# JavaScript Challenges Fun App 🚀

A fun React project filled with mini JavaScript coding challenges and exercises.
This app was built to practice core JavaScript concepts in a creative and interactive way.

---

## ✨ Features

This project includes multiple JavaScript mini-games and exercises such as:

* 😱 Panicify String
* 🤫 Whisper Function
* 🔀 Random Capitalization Generator
* 😀 Emojify Text
* 🧩 Anagram Checker
* 🔁 Flatten Nested Arrays
* 🔢 Letter Frequency Counter
* 🍬 Filter Sweet Candies
* 🎙️ Get Free Podcasts
* 🏆 Podcast Awards Generator
* 🌸 Keep the Flowers
* 📊 Vote Counter
* 🧠 Reduce(), Filter(), Map(), Sort() practice
* 🌐 API Fetching Exercises
* And many more...

# 🚀 Getting Started

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

# 🎯 Purpose of This Project

The goal of this project is to improve JavaScript problem-solving skills by building small interactive exercises instead of only reading theory.

Each component focuses on one JavaScript concept and turns it into a playable mini challenge.
